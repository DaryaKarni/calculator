# Calculator

Простой веб-калькулятор, реализованный на чистом JavaScript без использования внешних библиотек.
Проект настроен с использованием Webpack, ESLint, Prettier и Husky.

## Task

Инициализировать базовое JS приложение (без использования каких либо библиотек и фреймворков). Для этого приложения создать Webpack конфиг, который позволяет запускать это приложение с оптимизированным кодом. На выходе в билде дожно быть два файла: HTML и JS.

## How to run the app

Для запуска проекта понадобится установленный Node.js.

### 1. Установка зависимостей

Сначала установить все необходимые пакеты (Webpack, ESLint и т.д.):

npm install

### 2. Запуск в режиме разработки

Запускает локальный сервер с Hot Reload. Калькулятор будет доступен по адресу `http://localhost:8080`.

npm start

### 3. Сборка проекта

Создает оптимизированную production-версию приложения в папке `dist`.
После выполнения этой команды в папке `dist` появятся файлы `index.html` и `bundle.js`, которые готовы к деплою.

npm run build

### 4. Проверка кода

Запуск ESLint для проверки стиля кода и ошибок:
Также эта проверка запускается автоматически перед каждым коммитом благодаря Husky.

npm run lint

## Folder Structure

Краткое описание структуры проекта:

- **`dist/`** — папка с готовой сборкой проекта. Содержит минифицированный HTML и JS код
- **`src/`** — исходный код приложения
  - **`modules/`** — JS-модули с логикой
  - `operations.js` — чистые математические функции без использования `eval` и `Math`
  - **`styles/`** — CSS файлы стилей
  - `index.html` — шаблон HTML страницы
  - `index.js` — точка входа JS, связывающая логику, стили и DOM-элементы
- **`.husky/`** — конфигурация pre-commit для автоматического линтинга
- **`webpack.config.js`** — настройки сборщика Webpack
- **`eslint.config.mjs`** — настройки линтера
